# Currency Swap Form - Setup Instructions

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Installation

1. Navigate to the problem2 directory:
```bash
cd src/problem2
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

Start the development server:
```bash
npm run dev
```

The application will open automatically in your browser at `http://localhost:3000`

## Building for Production

To build the application for production:
```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

To preview the production build:
```bash
npm run preview
```

## Features

- Real-time exchange rate calculation
- Bidirectional amount input (send or receive)
- Token icons displayed next to currency names
- Form validation with clear error messages
- Loading states for async operations
- Modern, responsive UI design
- Smooth user experience with proper feedback
